v1.42.0 May 6, 2026 Feature Live Sync
Live model sync — auto-remove deprecated models
Live model sync (new base capability)
  • BasePricingProvider._fetch_live_model_ids() — queries each provider's /v1/models API on a 6-hour cache; no extra requests on every call
  • _apply_live_filter() — removes models absent from the live list; exact + prefix matching handles versioned aliases (e.g. gpt-4o ↔ gpt-4o-2024-11-20)
  • Conservative fallback — if filtering would empty the list (naming mismatch), serves full static list and logs a warning
  • Deprecated models are logged at INFO level: [OpenAI] Removed 2 deprecated model(s): [...]
