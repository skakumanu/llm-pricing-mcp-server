Widget Attributes

    

      
        

          Attribute
          Default
          Description
        

      
      
        

          data-models
          (all)
          Comma-separated model names to display. Omit to show all models from all providers.
        

        

          data-provider
          (all)
          Filter to a single provider (e.g. openai, anthropic). Ignored when data-models is set.
        

        

          data-theme
          dark
          dark or light
        

        

          data-layout
          table
          table for a full pricing table, compact for a card grid.
        

        

          data-refresh
          300
          Seconds between auto-refresh. Set to 0 to disable.
        

        

          data-server
          (auto)
          Override the server origin. Defaults to the hostname from which widget.js was loaded.
        

      
    

  


  
  

    
Public API Endpoint

    

      The widget fetches from a public endpoint — no API key required.
    

    

      GET
      /pricing/public
      
Returns simplified pricing for embedding. No authentication needed.

    

    
Query parameters:

    

      
models — comma-separated model names (optional)

      
provider — filter to one provider (optional)
