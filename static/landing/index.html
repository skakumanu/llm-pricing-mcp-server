Sign up free

Tell the router your max cost per 1M tokens, minimum quality score, or preferred provider.

3

Get routed automatically

# Find the best model under $2/1M tokens with quality ≥ 70

MCP Integration

Use from Claude, VS Code & Cursor

Connect any MCP-compatible AI assistant to your LLM Pricing server in under 2 minutes.

Step 1 — Clone & install
git clone https://github.com/skakumanu/llm-pricing-mcp-server.git
cd llm-pricing-mcp-server
pip install -r requirements.txt
Step 2 — Add to your client config
📁 Windows: %APPDATA%\Claude\claude_desktop_config.json 📁 Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "llm-pricing": {
      "command": "python",
      "args": ["-u", "mcp/server.py"],
      "cwd": "/path/to/llm-pricing-mcp-server",
      "env": { "PYTHONPATH": "/path/to/llm-pricing-mcp-server", "PYTHONUNBUFFERED": "1" }
    }
  }
}
📁 Open Cline in VS Code → Settings (⚙️) → MCP Servers → Edit JSON
